# Customer Retention & Churn Analysis Dashboard

## Overview
This Power BI dashboard analyzes customer retention and churn patterns using the Telco Customer Churn dataset.

## Key Insights
- Churn Rate: 26.5%
- Retention Rate: 73.5%
- Male and Female customers are almost equally distributed
- Fiber Optic is the most used internet service
- Electronic Check customers show higher churn
- Two-year contract customers contribute higher total charges

## Tools Used
- Power BI
- Excel

## Dataset
Telco Customer Churn Dataset
